### *(If you have not installed Node.js, please install it by [clicking here](https://nodejs.org/en).)*

# If you are running code for the first time

### Install
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
